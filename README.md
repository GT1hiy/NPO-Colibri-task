# NPO-Colibri-task

Необходимо написать приложение калькулятор (самое простое) со скрытыми меню: 

• Необходимые функции (сложение, вычитания т.п.) отображены на изображение. 

• Возможность операций с числами до 25 знаков. 

• При долгом нажатие (4 секунды) на =, а после (в течение 5 секунд) ввода комбинации «123» должен открыться 2 экран с одной надписью “Секретное меню” и кнопкой назад Среда разработки qt, технологии qml для графики!
